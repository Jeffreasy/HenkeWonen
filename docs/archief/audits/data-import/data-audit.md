## Import-risico's

- .xls support is verplicht; Interfloor is legacy Excel en moet via parser/conversiepad blijven werken.
- 460 codecellen kwamen als numeriek uit Excel; import moet codes expliciet naar string serialiseren.
- 20 codevoorbeelden beginnen met een punt; trimming/number-casting zou deze beschadigen.
- 95 prijskolom-observaties hebben onbekende btw-modus; import-preview moet mapping bevestiging vragen.
- 24 sheets bevatten vermoedelijke sectierijen; sectionLabel/subcategory moet doorlopen naar volgende productregels.
- 4 bestandsnamen komen op meerdere locaties voor; deduplicatie mag niet op bestandsnaam gebeuren.
- 58 logistieke/verpakkingskolom-observaties gevonden; import moet deze scheiden van productPrices.
